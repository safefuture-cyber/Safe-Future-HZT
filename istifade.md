# Saytdan istifadə qaydaları
<p class="kicker">Saytdan düzgün və təhlükəsiz istifadə üçün qısa bələdçi.</p>

<div class="section">
  <h2>Qısa təlim videoları</h2>
  <ul>
    <li>Profil gizliliyi necə qurulur? — <em>tezliklə</em></li>
    <li>Şikayət etmək addımları — <em>tezliklə</em></li>
  </ul>
</div>

<div class="section">
  <h2>“Necə məlumat verilir?”</h2>
  <ul>
    <li>Hadisəni qısa yaz: nə oldu, nə vaxt oldu, harada oldu?</li>
    <li>Dəlil saxla: ekran görüntüsü, linklər, istifadəçi adları.</li>
    <li>Təcili təhlükə varsa: dərhal uyğun təcili xəttlərə müraciət et.</li>
  </ul>
</div>
