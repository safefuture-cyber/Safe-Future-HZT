# Haqqımızda
<p class="kicker">Layihənin məqsədi, komandası və əməkdaş qurumlar.</p>

<div class="section">
  <h2>Missiya və vizyon</h2>
  <ul>
    <li><strong>Missiya:</strong> Gəncləri kiber təqib risklərindən qorumaq üçün maarifləndirmə və dəstək.</li>
    <li><strong>Vizyon:</strong> Təhlükəsiz, etik və məsuliyyətli rəqəmsal mühit.</li>
  </ul>
</div>
