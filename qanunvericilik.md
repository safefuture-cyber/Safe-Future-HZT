# Əlaqəli qanunvericilik
<p class="kicker">Bu bölmədə kiber təqib/kiberbullinq hallarında ümumi hüquqi istiqamət və müraciət yolları toplanır.</p>

<div class="section">
  <h2>Nələr olacaq?</h2>
  <ul>
    <li>Azərbaycan üzrə əsas hüquqi çərçivə (ümumi izah)</li>
    <li>Şikayət/müraciət edilə bilən qurumlar</li>
    <li>Məxfilik və sübutların qorunması üçün tövsiyələr</li>
  </ul>
</div>
